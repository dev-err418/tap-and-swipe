# How to use this template

This file is a reference. To start a real draft, run:

```bash
npx tsx scripts/new-draft.ts "Working title"
```

That writes `content/drafts/{8-hex}.mdx` with the same shape as below — the random ID keeps the URL unguessable. Edit the file, share `/drafts/{id}` for review (noindex, not in sitemap/RSS), then publish:

```bash
npx tsx scripts/publish-draft.ts {id} {final-slug} --pair   # typical: episode + case study
npx tsx scripts/publish-draft.ts {id} {final-slug}          # episode only
npx tsx scripts/publish-draft.ts {id} {final-slug} --case-study
```

The publish script runs the Discover readiness check and rolls back on failure. Empty-string fields are stripped before writing the published file, so leave anything you don't need blank rather than deleting the line.

See `seo/discover-playbook.md` for headline/image guidance.
